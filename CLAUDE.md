# Claude Code Configuration

## Build & Development Commands

```bash
npm install              # Install all workspace dependencies
npm run dev              # Start dev server with local proxy
npm run build            # Production build (static export)
npm run lint             # ESLint check
npm run prettier         # Format .tsx files with Prettier
npm test                 # Run Jest tests (discovery package)
npm run discovery        # Regenerate API config from Discovery.yml
npm run circular         # Check for circular imports
```

## Pre-Commit Checks

Before committing changes, run:
1. `npm run lint` — must pass with zero errors
2. `npm test` — must pass if `packages/discovery/` was modified
3. `npm run discovery` — must run if `Discovery.yml` was modified (regenerates `packages/common/config/`)

## Notes

- Do NOT edit files in `packages/common/config/` — they are generated by the discovery pipeline
- This repo uses PatternFly **5** (not 6) — use `pf-v5-u-*` CSS utility classes
- Static export mode — no server-side features (API routes, middleware, ISR)
